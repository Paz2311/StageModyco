# StageModyco

Ce dépôt comporte des scripts faits dans le cadre de deux stages au Laboratoire MoDyco, qui ont porté sur le traitement de corpus pour l'analyse de l'interface syntaxe-prosodie en français, dans le corpus Rhapsodie, et dans le pidgin nigérien Naija. 

Vous trouverez un dossier correspondant au corpus de chaque langue/pidgin, ainsi que le dossier _docs_, dédié aux descriptions des fonctions de chaque script.
